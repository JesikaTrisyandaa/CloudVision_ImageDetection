CloudVision: AI untuk Klasifikasi Awan

Project ini mengimplementasikan metode Convolutional Neural Network (CNN) dalam proses klasifikasi citra awan. Model dilatih untuk mengenali karakteristik visual dari berbagai jenis awan sehingga mampu memberikan hasil identifikasi secara otomatis berdasarkan input gambar.

  • Klasifikasi Citra

    Mengidentifikasi tipe awan berdasarkan karakteristik visual pada citra digital.

  • Real-Time Processing

    Proses klasifikasi dilakukan dengan cepat sehingga hasil identifikasi dapat ditampilkan secara langsung.

  • Deep Learning-Based

    Model dilatih menggunakan Convolutional Neural Network (CNN) untuk mengenali pola visual awan dengan lebih akurat.

Metodologi

Bagaimana CloudVision Bekerja

01

Pengumpulan Dataset

Menggunakan dataset gabungan dari empat dataset publik (CCSN, GCD, Cloud Photos Mendeley, dan Cloud-ImVN 1.0) yang menghasilkan 18.980 citra awan dalam 11 kelas berdasarkan klasifikasi WMO.

02

Preprocessing

Citra diresize menjadi 224 × 224 piksel, kemudian dataset dibagi menjadi training, validation, dan testing. Data training selanjutnya diaugmentasi menggunakan rotasi, zoom, pergeseran, dan horizontal flip.

03

Training Model CNN

Arsitektur transfer learning berbasis MobileNet dengan optimizer Adam, learning rate 1e-4, batch size 22, serta menggunakan Early Stopping untuk memperoleh model dengan performa terbaik.

04

Evaluasi & Deployment

Model dievaluasi menggunakan confusion matrix, accuracy, precision, recall, dan F1-score, kemudian di-deploy melalui REST API berbasis Flask untuk diintegrasikan ke dalam aplikasi web.
